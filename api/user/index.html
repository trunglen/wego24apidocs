User Api Docs

User API

User Profile includes:

- user_id: string
- phone: string
- pverified: boolean
- fb_id: string
- fb_token: string
- fname: string
- lname: string
- email: string

Danh sách User API:

GET /api/user/profile
POST /api/user/profile
POST /api/user/send_phone_code
POST /api/user/verify_phone
POST /api/user/change_password
POST /api/user/avatar

Get User Info

GET /api/user/profile

Response Data: <user profile>

Update User Info

POST /api/user/profile

Body:

{
    "phone": "*091-----",
    "email": "*nguyenvana@gmail.com",
    "password": "*",
    "fb_id": "",
    "fk_token": "",
    "fname": "*Văn A",
    "lname": "*Nguyễn"
}

Response: <default>

Send Code

POST /api/user/send_phone_code

Body:

{
    "phone": "091234234"
}

Response: <default>

Verify Phone Number

POST /api/user/verify_phone

Body:

{
    "phone": "091234234",
    "code": "123456"
}

Response: <default>

Đổi mật khẩu

POST /api/user/change_password

Body:

{
    new_password: "12345678",
    old_password:" 123456"
}

Response: <default>

Avatar

GET: GET /api/user/avatar

UPLOAD: POST /api/user/avatar

Body: form-data cho avatar Response data : null
