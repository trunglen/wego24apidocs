Shipment API

New Shipment

POST /api/shipment/new?access_token=<access_token>

  • Currency: “dong”, “usd”
  • Payment At: “picked_up”, “delivered”, “other”
  • After: timestamp
  • Before: timestamp

Update Shipment

POST /api/shipment/update/:id Request body: <shipment data> Response data: <shipment object>

Delete Shipment

POST /api/shipment/delete/:id Response data: null
