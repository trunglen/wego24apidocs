Wmap Api Docs

Carrier Socket

Connect to ws://<wmap_host>/wmap/carrier/join?access_token=<access_token>

Example: ws://203.162.123.63:3002/wmap/carrier/join?access_token=MHJNaali5bJvbYb49jGRo97QQFSt636Kh9JRKlSWojW3vobWZVt4GN3QtlZLUUcvpmHCUN7w9USazD96

Echo (for Test/Ping)

Request: /echo <data>

Response: /echo <received data>

Example: /echo 123 -> /echo 123

Set Carrier location

type Point = {"lat": <float>,"lng":<float>,"time":<time stamp>}

Request: /carrier/at <Array of Point>

Các Point sắp xếp theo thứ tự tăng dần thời gian. Như vậy, Point cuối cùng chứa vị trí mới nhất

Response: /carrier/at <len of recieved locations

Example: /carrier/at [{"lat":10,"lng":20,"time":14235346456},{"lat":10.01,"lng":20,"time":14235346460}] -> /carrier/at 2
